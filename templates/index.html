COMPRESSION CRATE
BY-NABHANYU JOSHI

Compress, restore, and verify files with an archive format that shows its work.

A full-stack build with a native C++ Huffman engine, SHA-256 integrity checks, Flask orchestration, and a drag-and-drop interface shaped more like an editorial control room than a standard dashboard.

Method

Huffman coding with native archive packing and byte-level reconstruction.

Integrity

Each archive carries a SHA-256 fingerprint for restoration checks.

Interface

Browser upload flow with result telemetry, ratio views, and downloads.

Archive operations

Compress

Generate a `.huf` archive with symbol frequencies and a checksum baked into the file.

Restore

Decompress archives and verify the restored file against the original SHA-256 digest.

Inspect

Surface ratios, dominant symbols, and integrity outcomes so the algorithm stays visible.
